# Email Reminder App

A simple, browser-based reminder app that sends you notifications at scheduled times.

## Features
- Create reminders with custom text, date, and time
- Browser notifications when reminders are due
- Reminders stored locally on your device
- Clean, modern interface

## How to Use

1. Open the app in your browser
2. Click "Enable Notifications" when prompted
3. Fill in the reminder form:
   - What you want to be reminded about
   - Date and time for the reminder
4. Click "Set Reminder"
5. Keep the browser tab open to receive notifications

## Hosting

This app works entirely in your browser and can be hosted on any static hosting service:

### Option 1: Netlify (Recommended)
1. Create a free account at [netlify.com](https://netlify.com)
2. Drag and drop the project folder to deploy

### Option 2: GitHub Pages
1. Push this code to a GitHub repository
2. Enable GitHub Pages in repository settings

### Option 3: Local Testing
Simply open `index.html` in your browser to test locally.
